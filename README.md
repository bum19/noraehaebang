# 주요 기능

-   webRTC와 활용한 실시간 화상 노래방 환경 제공
-   2가지 노래방 모드
    -   일반 노래방
        -   MR과 가사 제공
        -   녹화
    -   퍼펙트 스코어
        -   사용자의 음성 데이터 분석
        -   분석된 데이터로 음정 평가
-   노래방 부가 기능
    -   인원제한, 비밀번호 설정
    -   추방
    -   채팅
    -   입력 변경
    -   종료 후 피드 작성
-   실시간 알림
-   녹화된 영상을 sns 피드 형태로 공유
-   DM
    -   방생성
    -   초대
    -   실시간 채팅
